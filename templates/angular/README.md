# Angular Template

This folder contains the official Angular application template for the Childcare Management ERP project.

## Purpose

This template is used to create or align Angular frontend applications with the project standards.

It defines:

- Angular version baseline
- TypeScript configuration
- ESLint configuration
- Prettier configuration
- EditorConfig
- VS Code recommendations
- common npm scripts

## Target applications

This template applies to:

- ChildcareManagementERP.Staff.Web
- ChildcareManagementERP.FamilyPortal.Web
- future Angular frontend applications

## Current baseline

- Angular 22
- Standalone components
- Signals
- New Angular control flow
- SCSS
- Strict TypeScript
- ESLint
- Prettier
- OnPush by default

## Usage

A new Angular application should first be generated with Angular CLI.

Then the configuration files from this template should be applied to the generated project.

Some template files may contain placeholders such as:

```text
__PROJECT_NAME__
__PROJECT_PREFIX__
```

These placeholders must be replaced with the actual project values.
